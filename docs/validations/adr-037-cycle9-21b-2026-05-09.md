**The p50 +44 % is not a regression.** It is the consequence of cycle 9 completing 44 *more* queries (588 vs 544) — those 44 queries that moved from `timeout` → `completed` are by definition slow-but-completable (30–60 s range) and pull the median up while p75–p99 *drop* because the upper percentile slots are now occupied by faster queries. The same composition effect drives the wall-clock saving: 44 queries × ~60 s saved per timeout = ~44 min, matching the 46 min total wall-clock saving.

The query-side substrate code is byte-identical 1.7.47 → 1.7.50 (the changes are bulk-load only). The improvement comes from on-disk layout / page-cache state differences between cycle 9's substrate and cycle 8's. Substrate parser/grammar/cancellation behavior is stable: 0 failures across 2,398 queries (cycle 8 + cycle 9 combined).

## What this validates structurally

- **ADR-037's prediction-from-instrumentation methodology.** Cycle 8's instrumented per-spill timings predicted ~5 h parser saving at 21.3 B. Cycle 9 measured 4 h 57 m. Match within statistical noise. The methodology is sound; future ADRs can use the same pattern (cycle-N instrumentation predicts cycle-N+1 wins, validates by direct measurement).
- **Substrate identity preservation across pipelined-spill change.** Cycle 9 produced byte-identical indices to cycle 8, despite the parser-side ownership-transfer concurrency. The architectural change is correctness-preserving at production scale.
- **The cycle-compression methodology (ADR-038 + ADR-039 in cycle 10) is now warranted.** Cycle 9's clean prediction-vs-measurement confirms we have the instrumentation to attribute multi-fix contributions in a single cycle.

## What this does NOT validate

- Cognitive profile is still unmeasured against the 1.7.50 substrate. The cognitive-profile-validation-drought (`docs/limits/cognitive-profile-validation-drought.md`) persists.
- ADR-038 (merge-phase read-side) and ADR-039 (MPHF) hypotheses are still projections. Cycle 10 closes those.
- The merge-phase three-regime structure is still measurable but unmitigated. ADR-038 targets it directly.
