# Mercury Examples

Runnable examples demonstrating Mercury's storage and temporal query APIs.

## Running

```bash
dotnet run --project examples/Mercury.Examples -- [storage|temporal|demo|all]
```

## Examples

### Storage (`storage`)

Basic file storage, multi-index queries, bulk loading, persistence and recovery.

### Temporal (`temporal`)

Valid-time ranges, time-travel queries, version tracking, bitemporal corrections, snapshot reconstruction.

### Demo (`demo`)

TB-scale capability overview, architecture details.
